# kw-blog-client

A Vite + Vue3 webapp to serve blog content of myself

- Production: http://www.kenji-wilkins.blog/
- StoryBook: https://kenjiwilkins.github.io/kw-blog-client/

# Architecture

- Build Tool -> Vite
- UI Library -> Vue3
- Deploy platform -> heroku
- Backend -> Express
- API -> contentful
- Monitoring -> Sentry
- Unit test -> Vitest + vue test utils (TBD)
- E2E Cypress (TBD)

## Getting Started

### install packages

```
yarn
```

### run dev server

```
yarn dev
```

### build

```
yarn build
```

### serve

```
yarn serve
```
